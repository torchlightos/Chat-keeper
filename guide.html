Official Guide

The Gold File

Follow these steps to get your conversations.json file from ChatGPT.

1

Request Export

Log in to ChatGPT. Go to Settings > Data Controls and click the Export button.

2

Check Email

Open the email from OpenAI. Click Download data export. You will get a .zip file.

3

Unzip Your File

PC / Mac

Right-click the zip file and choose Extract All (Windows) or double-click (Mac) to see the folder inside.

iPhone (iOS)

Open the Files app. Tap the .zip file once. A folder will instantly appear next to it.

Android

Open Files by Google. Tap the zip file in Downloads, then tap Extract.

4

Upload to Keeper

Inside the folder, find conversations.json. This is the only file you need to upload to Chat Keeper.

No data ever leaves your device
